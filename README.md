# Modern Calculator App

A modern calculator with a clean UI and multiple color theme modes.

![React](https://img.shields.io/badge/React-19-61DAFB?style=for-the-badge&logo=react&logoColor=black) ![TypeScript](https://img.shields.io/badge/TypeScript-5.8-3178C6?style=for-the-badge&logo=typescript&logoColor=white) ![Vite](https://img.shields.io/badge/Vite-7-646CFF?style=for-the-badge&logo=vite&logoColor=white) ![ESLint](https://img.shields.io/badge/ESLint-9-4B32C3?style=for-the-badge&logo=eslint&logoColor=white) ![Color Themes](https://img.shields.io/badge/Color_Themes-Multiple-FF6B6B?style=for-the-badge&logo=palette&logoColor=white)

## Features

- Standard arithmetic operations (+, -, x, /)
- Clean, modern UI design
- Multiple color theme modes
- Responsive layout
- Built with React 19 and TypeScript for type safety

## Tech Stack

| Technology | Version |
|------------|---------|
| React | 19.1 |
| TypeScript | 5.8 |
| Vite | 7.1 |
| ESLint | 9 |

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)

### Installation

```bash
npm install
```

### Run Dev Server

```bash
npm run dev
```

### Build

```bash
npm run build
```
